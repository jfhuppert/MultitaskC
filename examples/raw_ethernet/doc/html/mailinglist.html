Mailing Lists

For Etherboot user issues there is a users mailing list. For Etherboot development issues there is developers mailing list. Unless a posting has to do with developing the software, it belongs on the user list. Archives of the lists are available, follow the links for more details. Please do not email me Etherboot related questions, I will not reply directly. Use the mailing lists instead. See the explanation for this policy.

If your question relates to LTSP, the LTSP mailing lists can be found by following the links from the LTSP home page.
